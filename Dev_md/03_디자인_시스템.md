## 테마 시스템

### 다크 모드
- **자동 모드**: 오전 6시~오후 6시 → 라이트, 이외 → 다크
- **수동 전환**: auto → light → dark → auto 순환
- `data-theme` HTML 속성으로 제어
- 쿠키(`theme_mode`)에 365일간 저장

### 컬러 테마
- `data-color` HTML 속성으로 제어
- CSS Custom Properties 오버라이드 방식
- 쿠키(`color_theme`)에 365일간 저장

## Glassmorphism 효과

```css
--glass-bg: rgba(255, 255, 255, 0.85);
--glass-border: rgba(255, 255, 255, 0.3);
--glass-blur: 12px;
```

네비게이션 바에 적용되며, 스크롤 시 불투명도가 높아집니다.
