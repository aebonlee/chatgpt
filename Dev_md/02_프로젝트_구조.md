## 라우팅 구조

| 경로 | 컴포넌트 | 설명 |
|------|----------|------|
| `/` | Home | 메인 페이지 |
| `/chatgpt-basics` | ChatGPTBasics | ChatGPT 기초 가이드 |
| `/prompt-engineering` | PromptEngineering | 프롬프트 엔지니어링 |
| `/gpts-store` | GPTsStore | GPTs & GPT Store |
| `/api-usage` | APIUsage | API 활용 가이드 |
| `/advanced-features` | AdvancedFeatures | 고급 기능 가이드 |
| `/model-comparison` | ModelComparison | 모델 비교 |
| `*` | NotFound | 404 에러 페이지 |

## 코드 스플리팅

모든 페이지는 `React.lazy()`와 `Suspense`로 지연 로딩됩니다:

```javascript
const Home = lazy(() => import('../pages/Home'));
const ChatGPTBasics = lazy(() => import('../pages/chatgpt-basics/ChatGPTBasics'));
// ...
```

## Context Provider 구조

```
ThemeProvider          → 테마 모드(auto/light/dark) + 컬러 테마(5가지)
  └── LanguageProvider → 언어(ko/en) + 번역 함수 t()
        └── BrowserRouter
              └── PublicLayout (Navbar + Routes + Footer)
```
