# ChatGPT Master - 페이지 구성

## 홈 페이지 (`/`)

### 섹션 구성

1. **Hero Section** — 히어로 배경 + 배지 + 타이틀 + 설명 + CTA 버튼 + 캐러셀
2. **Features Section** — 4개 기능 카드 (체계적 학습, 프롬프트 마스터, API 개발, 실전 활용)
3. **Learning Paths** — 6개 학습 경로 카드 (사이트 설정에서 동적 생성)
4. **Stats Section** — 통계 (6+ 가이드, 40+ 주제, 6 카테고리)
5. **Workflow Section** — 4단계 학습 플로우
6. **CTA Section** — 하단 콜투액션
