# ChatGPT Master - 컴포넌트 가이드

## 공통 컴포넌트

### 1. Navbar (`src/components/layout/Navbar.jsx`)

**기능:**
- Glassmorphism 스타일의 고정 네비게이션 바
- 스크롤 시 배경 변경 (`scrolled` 클래스)
- 5가지 컬러 테마 피커
- 다크/라이트/자동 테마 토글
- 한/영 언어 전환
- 반응형 모바일 메뉴 (1024px 이하)

**네비게이션 항목:**
```javascript
const NAV_ITEMS = [
  { path: '/chatgpt-basics', ko: 'ChatGPT 기초', en: 'Basics' },
  { path: '/prompt-engineering', ko: '프롬프트 가이드', en: 'Prompt Guide' },
  { path: '/gpts-store', ko: 'GPTs', en: 'GPTs' },
  { path: '/api-usage', ko: 'API 가이드', en: 'API Guide' },
  { path: '/advanced-features', ko: '고급 기능', en: 'Advanced' },
  { path: '/model-comparison', ko: '모델 비교', en: 'Models' },
];
```

### 2. Footer (`src/components/layout/Footer.jsx`)

**구성:**
- 3열 그리드: 브랜드 설명 | 빠른 링크 | 연락처
- Family Site 셀렉트 박스 (DreamIT Biz, Teaching AI, Claude Master)
- 저작권 정보

### 3. CodeBlock (`src/components/CodeBlock.jsx`)

**기능:**
- 구문 강조 (Python, JavaScript, Bash, JSON, 범용)
- 복사 버튼 (Clipboard API + 폴백)
- 하이라이트 라인 (코멘트에 `highlight` 또는 `important` 포함 시)
- Dracula/One Dark 색상 스킴

**사용법:**
```jsx
<CodeBlock language="python" code={`def hello():
    print("Hello, World!")  # highlight
`} />
```

**지원 언어:**
| language 값 | 구문 강조 |
|------------|----------|
| `python`, `py` | Python |
| `javascript`, `js`, `jsx`, `ts`, `tsx` | JavaScript/TypeScript |
| `bash`, `sh`, `shell`, `zsh` | Shell |
| `json`, `jsonc` | JSON |
| 기타 | 범용 (키워드/문자열/주석) |

### 4. TipBox (`src/components/TipBox.jsx`)

**타입:**

| type | 색상 | 아이콘 | 용도 |
|------|------|--------|------|
| `tip` | 초록 | 전구 | 유용한 팁 |
| `warning` | 노랑 | 경고 삼각형 | 주의사항 |
| `important` | 파랑 | 정보 | 중요 정보 |
| `danger` | 빨강 | X 마크 | 위험/금지 |

**사용법:**
```jsx
<TipBox type="tip" title="선택적 제목">
  팁 내용을 여기에 작성합니다.
</TipBox>
```

### 5. FeatureCard (`src/components/FeatureCard.jsx`)

홈 페이지 기능 카드. 아이콘, 제목, 설명을 표시합니다.

```jsx
<FeatureCard icon="fa-graduation-cap" title="체계적 학습" description="..." />
```

### 6. HeroCarousel (`src/components/HeroCarousel.jsx`)

- 자동 슬라이드 (5초 간격)
- 하단 도트 네비게이션
- 페이드 인 애니메이션

### 7. SEOHead (`src/components/SEOHead.jsx`)

- `document.title` 동적 설정
- Open Graph 메타 태그 관리 (og:title, og:description, og:url 등)
