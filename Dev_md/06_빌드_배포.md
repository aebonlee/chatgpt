# ChatGPT Master - 빌드 & 배포

## 로컬 개발

### 설치

```bash
cd ~/chatgpt
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

- URL: http://localhost:5177
- Hot Module Replacement (HMR) 지원
- 자동 브라우저 열기

### 프로덕션 빌드

```bash
npm run build
```

- 출력 디렉토리: `dist/`
- 코드 스플리팅: vendor, 각 페이지별 청크
- 소스맵: 비활성화 (프로덕션)

### 빌드 미리보기

```bash
npm run preview
```

```javascript

  plugins: [react()],
  base: '/',
  server: {
    port: 5177,
    open: true
  },
  build: {
    outDir: 'dist',
    sourcemap: false,
    rollupOptions: {
      output: {
        manualChunks: {
          vendor: ['react', 'react-dom', 'react-router-dom'],
        },
      },
    },
  }
});
```

## 환경 변수

현재 Supabase 등 외부 서비스를 사용하지 않아 환경 변수가 필요하지 않습니다.
추후 Supabase 연동 시 `.env` 파일에 다음을 추가:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_anon_key
```

## SPA 라우팅 주의사항

React Router를 사용하므로 서버에서 모든 경로를 `index.html`로 리다이렉트해야 합니다.

### Nginx 설정 예시

```nginx
location / {
    try_files $uri $uri/ /index.html;
}
```

### Netlify (`_redirects` 파일)

```
/*    /index.html   200
```
